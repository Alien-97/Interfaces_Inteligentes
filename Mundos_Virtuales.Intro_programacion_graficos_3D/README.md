1.	Qué funciones se pueden usar en los scripts de Unity para llevar a cabo traslaciones, rotaciones y escalados. 

Rpta:	* Podemos trasladar(mover un objeto de lugar) un objeto  en cualquiera de sus tres ejes. En Unity contamos con la propiedad *Transform*, para trasladar un objeto tenemos que manipular las coordenadas en el campo *Position*, en Unity, el sistema de referencia más usado toma *x* como la .... para moverse a derecha o izquierda, ç*y* se considera la altura, y *z* la profundidad.

Para rotar un objeto, pues existe el campo *Rotation*, para inclinar un objeto, cambiar su persepectiva en cualquiera de los ejes.

Y por último, para los escalados, se usa el campo *Scale*, que sirve para modificar las dimensiones de un objeto.
